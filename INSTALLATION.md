# Установка

1. Загрузите содержимое репозитория на хостинг, убедитесь что корневой каталог указывает на `/public`.
2. Создайте базу данных MySQL 8 и пользователя с доступом.
3. Импортируйте `sql/schema.sql`, затем `sql/seed.sql` в созданную базу.
4. Скопируйте `.env.example` в `.env` и укажите параметры подключения к базе данных.
5. Убедитесь, что PHP 8.1+ с расширением PDO MySQL доступен, включены сессии.
6. Откройте сайт в браузере, перейдите на страницу входа `/public/login.php`.
7. Авторизуйтесь тестовым пользователем: `owner@example.com` / `password` и начните работу.
